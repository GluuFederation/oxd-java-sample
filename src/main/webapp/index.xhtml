oxd-java sample webapp

Welcome to oxd-java sample webapp

This app demonstrates how to integrate oxd-java* into a standard web application.

This app is split into 3 sections:

  1. A settings page to initialize communication with an oxd-server.
  2. A sample workflow that showcases authentication using OpenID connect.
  3. UMA

(*) oxd-java is a java lib that facilitates authentication of users against an OpenID Connect identity provider such as Gluu Server.
